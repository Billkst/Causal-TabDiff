# Agent Skills Reference

## Available Skills

### Project Skills
- **mlflow** - ML实验跟踪、模型注册版本管理、模型部署和实验复现
- **brainstorming-research-ideas** - 结构化研究方向探索框架
- **pdf** - PDF文件操作（读取、合并、分割、旋转、水印、表单填充等）
- **xlsx** - 电子表格文件处理（读取、编辑、创建、格式化、清理数据）
- **ml-paper-writing** - 撰写ML/AI学术论文（NeurIPS、ICML、ICLR等）
- **python-testing** - Python测试策略（pytest、TDD、fixtures、mocking）
- **continuous-learning-v2** - 读写instincts避免重复错误
- **docx** - Word文档操作（创建、编辑、格式化）
- **creative-thinking-for-research** - 研究创意思维认知科学框架
- **deployment-patterns** - 部署工作流、CI/CD、Docker容器化
- **context7-docs-lookup** - 获取最新库文档和API参考
- **aoti-debug** - 调试AOTInductor错误和崩溃
- **skill-creator** - 创建新技能的指南

### OpenCode Skills
- **playwright** - 浏览器自动化（测试、截图、网页交互）
- **frontend-ui-ux** - UI/UX设计和实现
- **git-master** - Git操作（atomic commits、rebase、历史搜索）
- **dev-browser** - 持久化浏览器自动化

## Usage Guidelines

在开始任何任务之前，检查是否有相关的skill可以使用：

1. **识别任务领域** - 确定任务属于哪个专业领域
2. **匹配技能** - 查找与任务领域重叠的skills
3. **加载技能** - 使用 `load_skills=[...]` 参数加载所有相关技能
4. **优先用户技能** - 用户安装的skills优先于内置默认技能

## Examples

```typescript
// ML实验跟踪
task(category="quick", load_skills=["mlflow"], ...)

// 学术论文写作
task(category="writing", load_skills=["ml-paper-writing"], ...)

// 前端UI工作
task(category="visual-engineering", load_skills=["frontend-ui-ux"], ...)

// 测试相关
task(category="quick", load_skills=["python-testing"], ...)
```
