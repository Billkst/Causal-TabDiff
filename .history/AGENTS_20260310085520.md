# Causal-tabdiff 项目规则
 
## 强制执行前置检查

在开始规划或执行任何任务之前，**必须优先检查 `skills` 目录**中是否存在可用skill。绝对禁止跳过此步骤直接执行任务或编写新代码。

## Conda 环境

本项目的conda环境为causal_tabdiff，启动命令为conda activate causal_tabdiff

## 语言强制规则

1. 你必须始终使用中文（简体中文）进行所有回复
2. 即使用户使用英文或其他语言提问，你也必须用中文回答
3. 技术术语、代码变量名、API名称可保留英文，但必须在首次出现时提供中文解释
4. 禁止在回复中混合使用其他语言进行解释（除非用户明确要求翻译）
