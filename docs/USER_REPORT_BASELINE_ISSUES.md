### 问题3: iTransformer/TimeXer 的 MSE/MAE 不是小数

**你的担忧**:
> "iTransformer 和 TimeXer 的 MSE/MAE 为什么都不是小数？这个结果怕是有大问题吧？是没归一化吗？"

**真相**: ✅ **你完全正确！预测目标不一致。**

| 模型 | 预测范围 | y_true 范围 | MSE | 问题 |
|------|---------|-----------|-----|------|
| iTransformer | [-101.98, 206.94] | [0, 1] | 1069.74 | ❌ 未归一化 |
| TimeXer | [-9.82, 28.60] | [0, 1] | 47.44 | ❌ 未归一化 |
| SSSD | [0, 1] | [0, 1] | 0.5263 | ✓ 正确 |
| SurvTraj | [0.32, 0.69] | [0, 1] | 0.2505 | ✓ 正确 |

**根本原因**:
- iTransformer/TimeXer 预测的是**原始特征值**（未归一化）
- SSSD/SurvTraj 预测的是**归一化风险值**（正确）
- 缺少 sigmoid 激活函数
- 输出维度错误

**技术细节**:
```
iTransformer 输出: (597, 7, 15) 应该是 (597, 7)
TimeXer 输出: (597, 105) 应该是 (597, 7)
```

**结论**: iTransformer 和 TimeXer 的 Layer2 结果**不可比较**，应该移除或重新训练。
